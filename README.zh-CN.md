## 🏗️ 项目架构

```
src/
├── app/                          # Next.js App Router 页面
│   ├── layout.tsx                # 根布局（Provider、导航栏、页脚）
│   ├── page.tsx                  # 首页（英雄区、推荐、最近浏览）
│   ├── products/                 # 商品列表 + 详情（SSG 预渲染）
│   ├── compare/                  # 产品对比页面
│   ├── checkout/                 # 结账流程（货到付款）
│   ├── orders/                   # 订单列表 + 详情 + Aramex 追踪
│   ├── profile/                  # 个人中心 + 地址管理
│   ├── theme/                    # 主题工作室
│   ├── trending/                 # 热门排行榜
│   ├── login|register/           # 登录注册（密码哈希）
│   ├── wishlist/                 # 收藏夹
│   └── api/shipping/             # Aramex 物流 API 代理
│
├── components/                   # 共享组件
│   ├── ui/                       # shadcn/ui 基础组件
│   ├── cart-provider.tsx         # 购物车状态 + localStorage
│   ├── user-provider.tsx         # 用户认证上下文
│   ├── cart-recommendations.tsx   # AI 购物车推荐
│   ├── compare-bar.tsx           # 浮动对比栏
│   ├── magnifier-zoom.tsx        # 桌面悬停放大镜
│   ├── price-chart.tsx           # recharts 价格走势图
│   ├── product-gallery.tsx       # 图片画廊 + 缩略图
│   ├── product-viewer-3d.tsx     # Three.js 3D 查看器
│   ├── review-section.tsx        # 评价系统
│   ├── search-dialog.tsx         # 全局搜索 + 自动补全
│   ├── stock-notify.tsx          # 到货通知弹窗
│   ├── theme-studio.tsx          # 主题定制界面
│   ├── navbar.tsx                # 导航栏
│   └── footer.tsx                # 页脚
│
├── hooks/
│   └── use-local-storage.ts      # 安全的 localStorage Hook
│
├── lib/
│   ├── data.ts                   # 静态商品数据（8 个产品）
│   ├── types.ts                  # 全局 TypeScript 类型
│   ├── crypto.ts                 # SHA-256 密码哈希
│   ├── env.ts                    # 环境变量验证
│   ├── recommendations.ts        # 余弦相似度推荐引擎
│   ├── review-store.ts           # 商品评价存储
│   ├── price-history.ts          # 模拟价格历史生成器
│   ├── compare-store.ts          # 对比列表
│   ├── theme-store.ts            # 主题持久化
│   ├── recently-viewed.ts        # 最近浏览追踪
│   ├── recent-searches.ts        # 搜索历史
│   └── shipping/                 # Aramex 物流适配器
│
├── i18n/                         # next-intl 配置
└── middleware.ts                 # 路由守卫（Session Token）
```

### 状态管理

```
UserProvider ─┬─> AddressProvider ──┬─> CouponProvider ──┬─> CartProvider
              │                     │                     │
              │                     │                     └─> WishlistProvider
              │                     └─> ...                                  
              └─> OrderProvider
```

所有状态使用 `useSyncExternalStore` 实现 hydration 安全、跨标签页同步的 localStorage 持久化。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建（使用全部 CPU 核心）
npm run build

# 启动生产服务器
npm start

# 运行测试
npm test

# 监听模式
npm run test:watch
```

### 环境变量

需要物流功能时创建 `.env.local`：

```env
ARAMEX_ACCOUNT_NUMBER=your_account
ARAMEX_USERNAME=your_username
ARAMEX_PASSWORD=your_password
ARAMEX_API_BASE=https://api.aramex.com/shipping/v1
```

不配置也能正常使用，只是 Aramex 物流追踪功能不可用。缺失的环境变量会在启动时输出警告。

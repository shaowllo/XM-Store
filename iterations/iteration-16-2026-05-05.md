## 本轮优化方向

### 界面设计优化（frontend-skill）

#### 1. 产品详情页沉浸式设计

**视觉方案**：
- 冷静克制的科技美学 — 大面积留白、高对比度排版、图片主导的视觉层次
- 12列网格布局：图片占7列，信息占5列，让产品图片成为绝对焦点
- 移除所有卡片化包裹，采用扁平化的信息层级

**内容规划**：
1. **Hero**: 产品主图全宽展示 + 缩略图导航 + 品牌标签 + 产品名 + 价格 + CTA
2. **Support**: 星级评分、颜色选择器（圆形带选中标记）、数量控制（圆形按钮组）
3. **Detail**: 服务保障横向排列（极速配送、正品保障、7天退换）
4. **Final CTA**: 全宽加入购物车按钮 + 收藏 + 分享

**交互设计**：
- 图片区域悬停轻微缩放 + 点击灯箱放大
- 颜色选择器带动画反馈（缩放 + 对勾标记）
- 数量控制使用圆形边框按钮组
- 滚动时产品信息区域渐入动画

**具体改动**：
- 图片区域改为 `aspect-[4/3]` / `aspect-square` 比例，更大更沉浸
- 移除所有 Badge 组件，改用简单的红色折扣标签
- 颜色选择器改为圆形按钮，选中时显示对勾
- 数量控制改为圆形边框按钮组
- 操作按钮改为 `rounded-full` 全圆角
- 添加英文标签 "Color" / "Quantity" 增加层次感
- 服务保障改为极简的图标+文字横向排列
- 相关产品区域添加 "You May Also Like" 英文标签

#### 2. 购物车抽屉视觉优化

**问题识别**：
- Navbar 中直接内嵌购物车 Sheet 逻辑，代码臃肿
- 购物车项使用卡片包裹，视觉过重
- 数量控制使用 Button 组件，过于厚重

**优化方案**：
- 提取独立组件 `CartSheet`，职责单一
- 移除卡片背景，采用扁平列表布局
- 数量控制改为轻量圆形边框按钮
- 移除按钮改为悬停显示，减少视觉噪音
- 空状态使用圆形图标背景
- 所有按钮改为 `rounded-full` 全圆角
- 添加 AnimatePresence 动画（进入/退出/布局）

**Navbar 重构**：
- 移除购物车相关 import 和逻辑（移至 CartSheet）
- Logo 背景改为纯色 `bg-foreground` 替代渐变
- 整体更简洁，职责更清晰

## 设计原则遵循

本轮界面优化遵循 frontend-skill 的核心原则：

1. **克制优先**: 移除卡片背景、Badge 组件、渐变效果，让内容本身说话
2. **强排版层次**: 使用字号、字重、间距、英文标签建立清晰的视觉层级
3. **一致的圆角系统**: 统一使用 `rounded-full` 操作按钮，`rounded-2xl` 图片区域
4. **减少装饰元素**: 移除不必要的图标标签、渐变背景、阴影效果
5. **动画服务体验**: 所有动画都有明确目的（图片缩放、悬停上浮、进度条指示）
6. **组件职责单一**: 提取 CartSheet 组件，Navbar 更简洁
